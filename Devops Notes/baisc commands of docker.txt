		STARTING DOCKER BASIC COMMMANDS
**UBUNTU**
docker pull ubuntu -> downloading ubuntu image
docker images-> listing all downloaded images
docker run ubuntu -> create and run container from ubuntu image
docker run -it ubuntu /bin/bash -> running in iterative mode i.e., in terminal
docker stop <conatiner_id> -> stop a runnnig container
docker rm <container_id> -> remove a container
docker rmi ubuntu -> delete the ubuntu image
docker system prune -a -> clean unused containers, images, and networks
docker system prune -a -volumes -> nuclear clean


**NGINX**
docker pull nginx
docker run -d -p 8080:80 --name mynginx nginx
docker ps
docker stop mynginx
docker start mynginx
docker rm -f mynginx
docker rmi nginx
